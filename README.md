# ximageLib
自带适配功能,适应所有的屏幕,相当于等比例缩放
可以定制标题栏和状态栏的背景色
可以定制返回按钮的图标
可以定制标题栏的文本和颜色
可以定制确定按钮的文本和颜色和drawbale背景
可以定制照相机图标
可以通过配置开启一系列的功能,包括
照相
裁剪
预览
选择图片数量为1的时候的效果
多张图片选择的效果
